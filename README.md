## Quick Start (Local Dev — No Docker)

### Prerequisites
- Java 17+
- Maven 3.9+
- Node 20+
- PostgreSQL 16 running locally

### 1. Database

```sql
CREATE DATABASE gurukulam_db;
```

### 2. Backend

```bash
cd backend
mvn spring-boot:run
```

On first boot the seeder auto-creates:
- **Admin account** → `username: admin` / `password: Admin@123`
- All 8 classes (Nursery → 5th) with Section A
- Academic year 2025-26 set as current

### 3. Frontend

```bash
cd frontend
npm install
npm run dev
```

App runs at **http://localhost:3000**  
Backend runs at **http://localhost:8080**  
(Vite proxies `/api/*` → backend automatically)

## Quick Start (Docker)

```bash
# From project root
docker-compose up --build
```

- Frontend → http://localhost:3000  
- Backend  → http://localhost:8080  
- Postgres → localhost:5432

## API Overview

### Public (no auth)
| Method | Path                  | Description      |
|--------|-----------------------|------------------|
| POST   | /api/auth/login       | Login            |
| GET    | /api/public/notices   | List notices     |

### Admin only
| Method | Path                                   | Description              |
|--------|----------------------------------------|--------------------------|
| GET    | /api/admin/dashboard/stats             | Summary stats            |
| GET/POST | /api/admin/classes                   | Manage classes           |
| GET/POST | /api/admin/academic-years            | Manage academic years    |
| GET/POST/PUT/DELETE | /api/admin/students        | Manage students          |
| POST   | /api/admin/students/{id}/reset-password | Reset student password  |
| GET/POST | /api/admin/teachers                  | Manage teachers          |
| POST   | /api/admin/teachers/{id}/reset-password | Reset teacher password  |
| POST   | /api/admin/teachers/{id}/assign-class   | Assign class to teacher |

### Teacher
| Method | Path                        | Description            |
|--------|-----------------------------|------------------------|
| GET    | /api/teacher/me             | Own profile            |
| GET    | /api/teacher/my-assignments | Assigned classes       |
| GET    | /api/teacher/my-students    | Students in my classes |

### Student
| Method | Path             | Description  |
|--------|------------------|--------------|
| GET    | /api/student/me  | Own profile  |
